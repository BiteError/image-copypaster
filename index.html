Paste or Open fills the active selection; with no selection it drops the image as a new canvas.

  • Drag on the canvas to select an area
  • Drag inside a selection to move it, drag a handle to resize
  • ↶/↷ Undo / Redo
  • ⇊/⇉ Flip selection
  • ↻ Rotate selection
  • ⛶/◯ Toggle rect / ellipse shape
  • Esc Cancel a pasted selection before committing it, or clear the current selection
  • ⋯ More tools: New, Select All, Roundness slider, Alpha Paste
  • Tap Pick 🎨, then tap the image to set the alpha-color key

Open a file or paste an image from clipboard to get started.
